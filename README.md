# 🛍️ E-Commerce Customer Behavior Analysis

## 📖 Project Overview

Understanding customer behavior is essential for every successful e-commerce business. This project analyzes customer purchasing patterns, spending habits, satisfaction levels, membership plans, and demographic information to uncover valuable business insights.

Using Python and data visualization techniques, this analysis helps businesses understand what drives customer spending, identify high-value customers, and improve customer engagement strategies.

The project demonstrates practical skills in **Data Analytics**, **Customer Behavior Analysis**, **Business Intelligence**, and **Data Visualization**.

---

## 🎯 Project Objectives

The main goals of this project are:

* Analyze customer purchasing behavior
* Measure overall business performance using KPIs
* Identify revenue-generating customer segments
* Evaluate membership program effectiveness
* Understand customer satisfaction trends
* Analyze spending patterns across cities and demographics
* Generate business insights for data-driven decision making

---

## 📂 Dataset Information

The project uses an **E-Commerce Customer Behavior Dataset** containing customer demographic details, purchasing information, spending habits, and satisfaction ratings.

### Dataset Features

| Column             | Description                     |
| ------------------ | ------------------------------- |
| Customer ID        | Unique customer identifier      |
| Gender             | Customer gender                 |
| Age                | Customer age                    |
| City               | Customer location               |
| Membership Type    | Membership plan                 |
| Total Spend        | Total customer spending         |
| Average Rating     | Average customer rating         |
| Satisfaction Level | Customer satisfaction category  |
| Discount Applied   | Whether a discount was applied  |
| Purchase Frequency | Frequency of customer purchases |

## 📈 Analysis Performed

### 1️⃣ Gender Distribution Analysis

Examines the proportion of male and female customers.

### 2️⃣ Membership Revenue Analysis

Evaluates revenue generated by different membership types.

### 3️⃣ City-Wise Revenue Analysis

Identifies cities contributing the highest revenue.

### 4️⃣ Customer Satisfaction Analysis

Analyzes customer satisfaction levels and engagement.

### 5️⃣ Age Distribution Analysis

Studies customer demographics across different age groups.

### 6️⃣ Discount Impact Analysis

Measures how discounts influence customer spending behavior.

### 7️⃣ Top Customer Analysis

Identifies the highest-spending customers.

### 8️⃣ Purchase Frequency Analysis

Examines how often customers make purchases.

### 9️⃣ Correlation Analysis

Explores relationships between numerical variables using a heatmap.

### 🔟 Spending vs Rating Analysis

Investigates whether spending behavior is related to customer ratings.
